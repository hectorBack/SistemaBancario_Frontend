# 💻 Sistema Bancario - Interfaz Web

Interfaz moderna y reactiva desarrollada con **Next.js**, diseñada para que los usuarios gestionen sus finanzas de forma intuitiva.

## 🚀 Tecnologías Utilizadas
* **Next.js** (App Router)
* **Tailwind CSS** (Estilos modernos)
* **Recharts** (Visualización de datos y gráficos circulares)
* **Lucide React** (Iconografía)
* **SweetAlert2** (Alertas y confirmaciones de usuario)

## 🎨 Características
* **Dashboard Dinámico:** Visualización en tiempo real de gastos por concepto y destino.
* **Formulario de Transferencias:** Validaciones estrictas de cuenta destino (8 dígitos), saldo disponible y monto máximo.
* **Diseño Responsive:** Optimizado para escritorio y dispositivos móviles.

## 🛠️ Configuración
1. Clona el repositorio.
2. Instala las dependencias: `npm install`.
3. Corre el proyecto: `npm run dev`.
